This is the frontpage of http://tiddlyspace.com maintained

# Contributing

This will be expanded on in due-course.

## Requirements

* Python
* pip
* Node
* tsapp

## Setup

    npm install
    sudo ./install_casperjs.sh

## Testing

First, in one terminal:

    tsapp serve

Second, in another terminal:

    grunt test
